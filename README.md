1.  **The Zealot Patch (Incentive Filter Update):** Previous versions assumed all actors were rational/selfish. v9.1 accounts for **Ideological Actors** (Zealots) who act against their own self-interest, making the "Kill Chain" more robust against political or religious volatility.
