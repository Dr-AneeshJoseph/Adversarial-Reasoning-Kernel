Prevents "Hallucination of Amnesia." The system scans the entire conversation history to lock in immutable constraints (Budget, Hardware, Geography) before processing new prompts.

### Phase 1: The Generator (Logic-as-Code)
Instead of writing prose, the AI maps the problem topology using **Python Pseudo-code**. This forces the model to define variables and logic gates, reducing "hand-wavy" hallucinations.
